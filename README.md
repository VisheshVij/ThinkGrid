# ThinkGrid
ThinkGrid is an AI-powered diagnostic web application designed to analyze and stabilize a user’s mental, physical, and cognitive state through a conversational interface. It operates as a structured chat system that guides users through stages such as intake, deep scan, analysis, and diagnosis. During this process, it gathers information about sleep, focus, energy, mood, physical health, and current stressors, then generates a report with insights, root causes, and actionable steps.

The project is evolving toward a more advanced approach called “Neuro-Adaptive Routing.” Instead of relying on fixed, time-based interactions, ThinkGrid shifts to a dynamic, state-driven model that adapts to the user’s cognitive bandwidth in real time. This is especially important for users with ADHD, autism, or burnout, where focus and energy can fluctuate unpredictably. The system tracks key states such as energy (high, medium, low), focus (clear, scattered, blocked), and sensory overload.

Based on these states, ThinkGrid adjusts its behavior. When the user has high bandwidth, it continues deeper questioning and analysis. When bandwidth drops, it simplifies interactions and reduces cognitive load. In low-energy or blocked states, it provides small, manageable actions instead of complex tasks. If the user becomes overwhelmed, the system enters an “emergency mode,” pausing the normal flow and offering minimal, stabilizing steps.

A central feature is the “shame-free loop,” which removes pressure when users miss tasks or disengage. Instead of creating backlog, the system resets and continues from the present moment, preventing further cognitive overload.

Technically, ThinkGrid combines a chat-based UI with Gemini AI and adaptive logic implemented in JavaScript. The goal is to transform it from a static diagnostic tool into a responsive system that supports users by adjusting to their real-time cognitive state.
